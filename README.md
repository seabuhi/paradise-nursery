# Paradise Nursery Shopping Application

## Project Overview
Paradise Nursery is an online plant shop web application built with React and Redux. 
It allows users to browse plants, add them to a shopping cart, and manage their orders.

## Features
- Browse plants by category (Succulents, Tropical Plants, Air Purifying)
- Add plants to shopping cart
- Update plant quantities in cart
- Remove plants from cart
- View total cost of cart items
- Responsive navigation bar

## Technologies Used
- React.js
- Redux Toolkit
- React-Redux
- CSS3

## How to Run
1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`
